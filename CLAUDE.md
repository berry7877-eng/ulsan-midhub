# 울산 미드허브 (ULSAN MidHub)

## 프로젝트 개요
울산 40~60대 중장년 취업희망자를 위한 온라인 통합 플랫폼. 소셜벤처.
청년 정책(~39세)에서 소외된 중장년의 취업정보 비대칭 문제를 기술로 해결.

**핵심 메시지:** "AI는 초급자의 일자리를 빼앗지만, 경험 많은 중장년에겐 오히려 기회다"
**포지셔닝:** "취업 플랫폼"이 아닌 "중장년의 삶을 다시 연결해주는 플랫폼"

## 기술 스택
- Next.js + TypeScript + Tailwind CSS
- Vercel 배포
- 공공데이터 API (고용24, 워크넷, 울산시 포털)
- 자동 크롤링 엔진 (예정)

## 핵심 기능 7가지
1. **정보 집약 대시보드** — 울산 7개+ 기관 공고 실시간 수집
2. **교육 통합 신청** — 폴리텍·직업훈련원 등 원스톱 신청
3. **정책 모니터링** — 울산시 일자리 부서 공고 자동 감지
4. **중장년 커뮤니티** — 취업 경험 공유, 정보 제보
5. **AI 활용 교육** — 대표자 직접 운영, 기초 무료 → 심화 유료 → 기업 출강
6. **건강 커뮤니티** — 대표자 직접 운영하는 조깅 번개 (오프라인 유입 채널)
7. **영상·음성 정보 제공** — TTS 읽어주기(Web Speech API), 주요 공고 숏츠화

## 타깃 사용자
- 울산 40~60대 취업희망자
- 청년 정책 탈락(40세 초과), 중장년 정책은 50대 중심 → 40대 중반~50대 초반이 가장 소외

## 수익 모델
| 단계 | 수익원 |
|---|---|
| 단기 (0~1년) | AI 교육 수강료, 건강·러닝 광고, B2G 공공 협력 |
| 중기 (1~2년) | B2B 채용 매칭, 교육기관 파트너십 |
| 장기 (2년~) | 타 지역 확장, SaaS 전환 |

- Next.js 개발 가능 (울산 잡허브 운영 중) → 개발비 0원
- AI 교육 직접 운영 가능
- 마라톤 현역 → 조깅 번개 직접 운영
- 포워딩 6년 경력 (중장년 취업 수요자 관점 보유)
